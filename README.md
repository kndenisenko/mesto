Четвёртый проект, выполненный в рамках яндекс.практикуме.
Проект выполнен по БЭМ - все файлы стилей разложены по своим папкам, в соотвествии с методологией. Используемые технологии: HTML (flex, grid), CSS, JS. Javascript используется впервые для проектов Практикума.
## Задача:
Сделать так, чтобы при нажатии на кноку редактирования профиля появлялся попап с формой. Первоначально в поля формы добавляются значения полей с именем и профессией. Попап позволяет их редактировать. Сохранение новых данных происходит по нажатию кнопки "сохранить" или по нажатию [Enter]. Данне "сохраняются" в разметке до перезагрузки страницы. При переполнении полей с именем и профессией вёрстка не должна ломаться, а текст, который не влез в поле, заменяется на "...".
