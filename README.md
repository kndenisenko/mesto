Четвёртый проект, выполненный в рамках яндекс.практикуме. В данном проекте впервые используется JS.
## Задача:
Сделать так, чтобы при нажатии на кноку редактирования профиля появлялся попап с формой. Первоначально в поля формы добавляются значения полей с именем и профессией. Попап позволяет их редактировать. Сохранение новых данных происходит по нажатию кнопки "сохранить" или по нажатию [Enter]. При переполнении полей с именем и профессией вёрстка не должна ломаться, а текст, который не влез в поле, заменяется на "...".
